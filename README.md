# 🎓 Student Record Management System

A simple **Python + SQLite** project to manage student records.

## 💡 Features
- Add new student
- View all students
- Update student details
- Delete a student
- Data stored in SQLite database

## 🛠️ Technologies Used
- Python 3
- SQLite3 (built-in database)
